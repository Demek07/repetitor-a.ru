Repetitor

Repetitor это Django-based проект для организации расписания лекций. Этот проект помогает организовывать группы, расписание, посещемость и оплаты.
Tech Stack

Client: HTML, CSS, JavaScript (Soft UI by Creative Tim)

Server: Django, SQLite
Функции

    Добавление/редактирование/деактивация студентов
    Управление группами.
    Добавление списка предметов.
    Добавление лекций в расписание. Вы также можете редактировать или удалять лекции.
    Возможность проверки посещаемости и оплаты определенных лекций.
    Предварительный просмотр для студентов, который автоматически рассчитывает процент посещаемости по заданному типу предмета.
